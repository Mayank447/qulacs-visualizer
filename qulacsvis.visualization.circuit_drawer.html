qulacsvis.visualization.circuit_drawer module¶

qulacsvis.visualization.circuit_drawer.circuit_drawer(output_method: Optional[str] = None, *, ppi: int = 150) → Union[str, PIL.Image.Image][source]¶

Draws a circuit diagram of a circuit.

Parameters

  • output_method (Optional[str]) – Set the output method for the drawn circuit. If None, the output method is set to ‘text’.

  • ppi (int) – The pixels per inch of the output image.
